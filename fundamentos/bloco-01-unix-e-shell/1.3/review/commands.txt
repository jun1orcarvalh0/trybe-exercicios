ls - ver o que tem dentro do diretório que está
ls * - especifica o que tem dentro dos diretorios que esta dentro do diretorio que está
mv - move e renomeia
cd - entra dentro dos diretórios
cd .. - volta para o diretório pai
touch - cria arquivos
mkdir - faz um novo diretório
rmdir - remove um diretório se ele estiver vazio
rm - remove arquivos
rm -rf - remove diretórios ignorando que eles estão cheios
cat - pega um arquivo e abre ele direto no terminal
less - abre uma página diferente no terminal para ler o arquivo
head - exibe os primeros itens do arquivo
tail - exibe os ultimos itens do arquivo
grep - pega palavras chaves do arquivo
wc - conta os numeros de linhas/caracteries/palavras
wc -l - conta o numero de linhas
wc -c - conta o numero de caracteries
wc -w - conta o numero de palavras
man - le a pagina sobre o comando no manual
whatis - breve descricao sobre o comando
apropos - fala quais comandos faz ea funcao
* - substitui varios caracteries
? - substitui um caractere
-i - ignora formatacao
-v - inverte os valores
-iv - ignora formatacao e inverte os valores
